ТРАССИРОВКА АВТОНОМНЫХ СИСТЕМ.

1. необходимые библиотеки из requirements.txt должны быть установленны.
2. запуск скрипта:
        py task1.py "www.yandex.ru"
        или
        task1.py "www.yandex.ru"
   если не введен IP/HOST, будет выброшено исключение:
        ValueError: PLEASE ENTER HOST NAME:
        >> task1.py 'www.yourhost.com'
3. Результат:

    что программа работает, а не упала.
